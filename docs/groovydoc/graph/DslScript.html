[Groovy] Class DslScript

        • graph.DslScript

        • Base script for using graph-dsl. This class may be used in scripts to make the dsl part of the script. 

          
 #!/usr/bin/env groovy

 @Grapes(
     @Grab(group='com.github.moaxcp', module='graph-dsl', version='0.20.0')
 )

 import graph.*
